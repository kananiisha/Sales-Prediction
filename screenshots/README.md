# 📸 Screenshots Guide

Add your application screenshots here. To generate these:

1. **Start the Flask app**:
   ```bash
   python app.py
   ```

2. **Navigate to**: `http://localhost:5000`

3. **Take screenshots of**:
   - `dashboard.png` - The main dashboard (after login)
   - `predict.png` - The prediction form page
   - `result.png` - A sample prediction result

4. **Save them in this folder** as:
   - `dashboard.png`
   - `predict.png`  
   - `result.png`

**Tips**:
- Use Chrome DevTools (F12) → Screenshot tool
- Or use Windows Snipping Tool (Win+Shift+S)
- Or use a screen capture tool like Snagit
- Crop to show just the relevant content (remove address bar if possible)

The README.md will automatically display these images when they're added to this folder.
